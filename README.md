# OneShot Reminder Releases

This public repository is the update channel for the [OneShot Reminder Android app](https://github.com/AzizjonKasimov/oneshot-reminder).

- Signed APK files are attached to entries on the [Releases page](https://github.com/AzizjonKasimov/oneshot-reminder-releases/releases).
- `version.json` tells installed copies of OneShot which release is current and where to download it.
- Application source code, build instructions, privacy details, and security guidance live in the main source repository.

Signing keys and signing properties are never stored here. Every APK must be signed with the same private key as the initial release so Android can verify in-place updates.
